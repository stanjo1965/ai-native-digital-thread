# 📖 쉬운 말 용어사전 (초등학생용)
> 어려운 말이 나오면 여기서 찾아보세요!

| 어려운 말 | 쉬운 뜻 | 한 줄 예시 |
|---|---|---|
| **AI 모델** | 똑똑하지만 매일 기억을 잃는 신입 | Claude·GPT·Gemini |
| **하네스(Harness)** | AI가 잘 일하는 ‘사무실 환경’ | 규칙·검사·팀을 묶은 것 |
| **CLAUDE.md** | ‘이렇게 해, 이건 하지 마’ 규칙 쪽지 | AI가 일 시작 전 항상 읽음 |
| **스킬(Skill)** | 자주 하는 일을 눌러주는 버튼 | ‘기획 문서 만들어줘’ 자동 |
| **에이전트(Agent)** | 한 가지 일 잘하는 AI 일꾼 | 설계·개발·검사 담당 |
| **훅(Hook)** | 규칙을 어기면 막아주는 자동 검사기 | 틀리면 저장이 안 됨 |
| **MCP** | AI가 바깥 도구를 쓰게 연결하는 것 | 메일·데이터베이스 연결 |
| **루프(Loop)** | 완성될 때까지 스스로 반복하기 | 만들고→검사→고치기 반복 |
| **랄프 루프** | 클로드 공식 ‘루프 엔진’ 플러그인 | 루프를 돌려주는 도구 |
| **완료 기준(종료 조건)** | ‘언제 끝낼지’ 정하는 규칙 | 테스트 다 통과하면 끝 |
| **Max Iteration** | 최대 몇 번까지 반복할지 | 예: 30번 넘으면 멈춤 |
| **PRD** | 무엇을 만들지 적은 목록 | 캘린더·CRUD·알람… |
| **CRUD** | 만들기·보기·고치기·지우기 | Create·Read·Update·Delete |
| **QA** | 잘 만들어졌는지 검사하는 일 | 검사 리포트 만듦 |
| **워크스페이스** | AI가 안전하게 작업하는 복사 공간 | 진짜 코드는 안 망가짐 |
| **커밋(commit)** | 작업을 ‘저장 도장’ 찍기 | 나중에 되돌리기 좋음 |
| **터미널** | 명령을 입력하는 검은 창 | 윈도우: PowerShell |
| **IDE / 편집기** | 파일을 만들고 고치는 프로그램 | 메모장·VS Code |
| **YAML** | 파일 맨 위에 이름·설명 적는 부분 | 스킬 파일 규칙 |

### 🌟 큰 그림 한 장
```
AI 모델(똑똑한 신입)
   └─ 하네스(사무실 환경)
        ├─ CLAUDE.md (규칙 쪽지)
        ├─ 스킬 (자동 버튼)
        ├─ 훅 (자동 검사기)
        └─ 에이전트 (AI 팀)
   └─ 루프(완성될 때까지 자동 반복)   ← 하네스 위에서 돈다
```
